# Zig Zag Background
A background using ziz-zag

## Tech Stack
- HTML
- CSS

## Instructions 
1. Clone the repo or download the zip file. 
2. Open index.html file in your browser.
3. You will a white background with a pink zig-zag background behind it and on top of it. Near the center of the page you will see an zig-zag across the page.
